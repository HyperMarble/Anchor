# Product Idea: Coding Prompt Optimizer

## What it is
Menu bar Mac app. Hyper focused on coding agents.

User types or speaks natural language coding prompt
→ auto formats to optimized structured prompt
→ works with ANY coding agent (Claude, Cursor, Copilot, Windsurf, etc.)
→ BYOK (bring your own API key)
→ better output every time

## Why menu bar
Developers live in their tools. Menu bar fits workflow perfectly.
No context switching. Always available. Stays out of the way.

## Pricing
- $7/month or $59/year
- BYOK — user brings own API key, no usage costs for us
- Pure subscription margin

## What makes it defensible
Not generic XML formatting. Coding specific:
- Detects language, framework, task type automatically
- Bug fix gets different structure than feature request
- Rust prompt structured differently than Python
- Learns which structures get best results per agent
- Gets smarter over time

## MVP scope
- Menu bar app (Mac first)
- Detect coding intent from natural language
- Format to optimized structure
- Copy to clipboard or inject directly into active app
- Works with any coding agent
- BYOK Claude API key
- $7/month or $59/year via Stripe
